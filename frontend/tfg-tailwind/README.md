# Frontend del Detector d’Aus — React + Vite + Tailwind CSS

Aquest frontend permet carregar imatges i vídeos, enviar-los al backend i visualitzar deteccions d’aus.

## Requisits
- Node.js 18+
- npm o pnpm

## Instal·lació
```bash
cd frontend
npm install
```

## Execució
```bash
npm run dev
```

El frontend estarà disponible a:
http://localhost:5173
